Leerdoel 1

Doel

Ik wil leren hoe ik animaties effectief kan inzetten om interactie en beleving in mijn project te verbeteren, zonder dat het afleidt of onnodig complex wordt.

Waarom

Animatie kan bijdragen aan gebruiksvriendelijkheid, feedback, sfeer en professionaliteit van een digitale ervaring. Het is een krachtig ontwerp- én ontwikkelmiddel.

Hoe ga ik dit aanpakken

- Ik ga animaties maken met CSS en/of JavaScript (bijv. transitions, keyframes, scroll-animaties). - Ik bestudeer bestaande voorbeelden en leer van frameworks als GSAP of Framer Motion (indien relevant).

Leerdoel 2

Doel

Alle websites die ik maak (mits nodig) mooi responsive maken zodat ze op elk apparaat goed werken.

Waarom

Als iedereen de website kan gebruiken, is het voor iedereen toegankelijk. Dit is belangrijk voor de gebruikerservaring en de toegankelijkheid van de website. Het is ook goed voor SEO.

Hoe ga ik dit aanpakken

- Beginnen met mobile-first design en media queries gebruiken om de website aan te passen aan verschillende schermformaten. - Gebruik maken van flexbox en grid om een responsieve lay-out te creëren.

Leerdoel 3

Doel

Een digitaal prototype kunnen maken met html css js om eventuele gebruikerstests te kunnen uitvoeren.

Waarom

Het is belangrijk om een prototype te maken zodat je kan testen of de website goed werkt en of de gebruikers het begrijpen. Dit kan helpen om problemen vroegtijdig op te sporen en op te lossen.

Hoe ga ik dit aanpakken

- Veel meekijken met klasgenoten en ervaren developers om te leren hoe zij prototypes maken.
